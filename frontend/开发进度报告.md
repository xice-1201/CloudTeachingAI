2. **P1.4 教师端** - 内容生产者，次优先
3. **P1.5 管理员端** - 运营管理，最后完成

### 开发顺序（P1.3 内部）
1. Dashboard + CourseList（快速看到效果）
2. CourseDetail + ResourceLearn（核心学习流程）
3. AbilityTest + LearningPath（AI 特色功能）
4. Assignment + Chat（辅助功能）
5. Notifications（通知中心）

### Mock 数据策略
- 在后端未就绪前，使用 Mock 数据开发
- 可以使用 MSW (Mock Service Worker) 或简单的 JSON 文件
- API 接口已定义，切换到真实后端只需修改 baseURL
